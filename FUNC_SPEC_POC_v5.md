# FUNC_SPEC_POC_v5.md

> 액화수소 인수기지 디지털 트윈 플랫폼 POC 기능명세서 (PRD)

---

## 1. 제품 목표

- 링크 1개로 참여기관이 접속하여 예상 플랫폼 모습을 체험
- 이벤트 중심 흐름이 끊기지 않고 모드 간 연계가 자연스럽게 동작
- 실데이터 없이 seed + mock으로 전체 흐름 재생 가능
- 설정/보고서까지 포함해 운영관리 느낌 제공

## 5. 수락 기준 (Acceptance Criteria)

### 전체 흐름
- 시나리오 시작 → 정상 → 이상 감지 → 이벤트 팝업 → 모드 전환 → SOP 실행 → 보고서 생성까지 끊김 없이 동작
- 어떤 시나리오에서든 관련 SOP가 1건 이상 추천됨
- 모든 버튼이 실제로 동작함 (작동하지 않는 버튼 없음)

### 3D
- GLB 로드 후 7개 설비 mesh가 정상 렌더링
- 이벤트 발생 시 trigger/affected 설비 컬러링이 3초 이내 반영
- 카메라 프리셋 이동이 부드럽게 동작

### 데이터
- 8개 시나리오 모두 에뮬레이터에서 재생 가능
- SC-08(하역 시나리오)에서 선박→로딩암→저장탱크 전주기 흐름 체험 가능
- phase 전환 시 적절한 알람/이벤트 발생
- 보고서 초안에 6개 데이터 소스(이벤트/KOGAS/KGS/KETI/세이프티아/SOP이력) 모두 포함
